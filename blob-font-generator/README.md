# Blob Font Generator

A Next.js 14 application for generating custom blob typography with organic, playful letter designs.

## Features

- **26 Custom Blob Letters**: All letters A-Z designed with consistent blob aesthetic
- **Thickness Control**: Scale letters from 1.0x to 2.0x thickness
- **Organic Variations**: Add rotation and vertical variance for playful effects
- **Grid Overlay**: Visualize letter structure with customizable grid
- **Full Customization**: Control colors, spacing, curvature, and more

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000/blob-font-generator](http://localhost:3000/blob-font-generator) in your browser.

## Build

```bash
npm run build
```

## Typography Design Principles

- All letters span y=5 to y=95 (same height)
- Horizontal bars: ~12 units tall with rounded ends
- Vertical stems: ~6 units wide
- Smooth bezier curves throughout
- Consistent organic blob aesthetic

## Controls

- **Display & Spacing**: Letter size, spacing, word spacing, vertical crop
- **Organic Variation**: Rotation and vertical variance for playful effects
- **Colors**: Letter and background colors
- **Blob Parameters**: Thickness, smoothness, curvature
- **Grid**: Overlay grid with configurable spacing and styling
